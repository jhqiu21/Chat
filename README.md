# Chat App

This is a Live Chat Software. You can communicate with other users the this chat room.

## Initial Setup

1. Make sure you have the following installed on your machine.
    
   - Go version 1.11+
   - npm
   - npx

1. Run Chat App Locally.

   - In the backend, run
     ```
     $ go run main.go
     ```
   - In the frontend, run
     ```
     $ npm start
     ```
1. Start your chat journey.
